# Sistema de Gestão de Portarias (SGP)

O **Sistema de Gestão de Portarias (SGP)** é uma aplicação web desenvolvida como parte de um projeto de pesquisa do **Instituto Federal Catarinense – Campus Videira**, com o objetivo de modernizar, automatizar e centralizar o gerenciamento de portarias institucionais.

O projeto surgiu a partir da evolução de um protótipo desenvolvido em **Google Sheets** e **Google Apps Script** durante um estágio curricular. A nova versão busca oferecer uma solução mais robusta, escalável e segura, permitindo o cadastro, gerenciamento, consulta e geração automatizada de portarias por meio de uma interface web intuitiva.

Além de atender às necessidades administrativas do IFC Campus Videira, o sistema também possui caráter acadêmico, servindo como estudo de caso para aplicação prática de conceitos de **Engenharia de Software**, **Desenvolvimento Web**, **Banco de Dados**, **Segurança da Informação** e **Arquitetura de Sistemas**, envolvendo estudantes do curso de Ciência da Computação.

---

## 🎯 Objetivos

- Automatizar o gerenciamento de portarias institucionais;
- Centralizar o armazenamento das informações em um banco de dados relacional;
- Facilitar a consulta pública e administrativa das portarias;
- Padronizar a emissão de documentos por meio de formulários;
- Disponibilizar um sistema seguro com diferentes níveis de acesso;
- Estruturar a aplicação para futuras integrações com sistemas institucionais, como o SIPAC.

---

## ✨ Funcionalidades

- Cadastro, edição e gerenciamento de portarias;
- Consulta pública de portarias;
- Busca avançada por número, ano, tipo, servidor e data;
- Controle de acesso com autenticação de usuários;
- Diferentes perfis de acesso (Administrador e Servidor);
- Geração automatizada de portarias padronizadas;
- Histórico e organização das publicações;
- Interface responsiva para desktop e dispositivos móveis;
- Estrutura preparada para futuras integrações institucionais.
